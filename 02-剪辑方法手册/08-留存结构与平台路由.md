# 留存结构剪辑

## 适用范围

面向信息流分发（抖音/小红书/视频号/B站等推荐流）的切片与短视频任务读取本文件；企业内训、完整教程、甲方定制等非信息流交付不强制。

本文件只管「从既有口述中选句、裁剪、搬移」的剪辑决策，不产钩子文案、不写口播稿——素材里没有的话不会被造出来，需要补录时标 `manual/confirm` 交用户。多数数值是经验参考线（非平台官方口径），以自有数据持续校正：作者 2026-07 实测基线是口播切片 2 秒跳出率平均约 45%——开头 2 秒是全片最贵的资产。

## 开头重构决策流

按顺序执行：

1. **第一刀删铺垫**：信息流切片下，开头的问候、自我介绍、背景铺垫（「大家好我是……」「今天想跟大家聊聊……」）属于无独有信息的整句删除，默认删；成片第一句必须是核心论点、结果或反直觉断言。若问候确实承载人格记忆点或用户要求保留，按 `operating-manual.md` A-roll 高风险闸门交确认——本条是「整句删除须证明保留段覆盖独有信息」在信息流场景下的预判定，不是绕开。
2. **口头钩子时长**：钩子句约 2–2.5 秒内说完、句内不留废气口（经验参考线）。超长就换更短的句，或只裁出句子最锋利的前半段（裁剪仍以完整语义单位为界）。
3. **钩子三公式路由**（从既有口述中**选句**，不是写句）：
   - **数字承诺**：真实数字 + 结果（「500 家门店的周报 10 分钟看完」）——教育/干货类默认档，数据/BI 类讲解优先在转写里找这类句子；
   - **模式打断**：反直觉断言、无铺垫直进行动（「别再手动导 Excel 了」）；
   - **提问开局**：制造信息缺口、观众自我代入（「你的会员数据为什么越算越乱？」）。
4. **爆点前置扫描**：通读剪后转写，重点排查 10–20 秒区间——多数观众撑不到那里（经验共识），把埋在中段的最强句/最强画面列为搬到开头的候选。**搬移 = 高风险重排**：只能整句（完整语义单位）搬移，逐个试听新接缝，按 `operating-manual.md` A-roll 高风险闸门确认，遵守 SKILL.md 红线「重排不造句」。
5. **录屏演示类变体**：可用「成果画面闪现约 0.3 秒再倒回正片」的冷开场——先让观众看到最终效果，再从头演示。
6. **宿主端可用官方句式**：ChatCut 对钩子前置有现成 prompt 模式（如 "Trim the clip so it starts with the most attention-grabbing sentence"），结果仍须按上述 1–4 逐项人工核对，不当黑盒。

首帧质检：首帧须有人脸、动势或高对比内容，禁止黑帧、转场中间帧、静止空镜开场；首帧构图按「可直接充当封面」的标准检查。竖版确需系列感时，标题页压到约 0.5–1 秒且与第一句口播同帧出现，不放切片编号或内部元信息。

## 钩子与兑现

「钩子与兑现成对」的执行细则：

- 钩子承诺的结果、数字或结论必须在成片**前 1/3 内**真实出现（内部默认值，超长片按比例或按用户确认调整）。
- 核对方法：把钩子句里的承诺对象（数字、结果、结论关键词）在剪后转写中定位实际出现的时间点；落在前 1/3 之外就把兑现段前移，或换更诚实的钩子句。
- 约束对象限于剪辑可控面：钩子片段的选择、搬移位置、包装标题与装饰文字。点击率与跳出率是一对制衡指标——钩子越强，兑现约束越硬。

## 中段留存

- **注意力时钟**：超过约 30–45 秒无任何变化（画面构图、语气、数据、直接对观众说话）的区间即「长时间视觉静止」信号——命中此参考线即视为信号成立，可选轻量变化：punch-in 缩放 15–25%、关键词字卡、2–4 秒 B-roll。仍禁止无任务堆动效，证据规则不松动。
- **开环句保护**：成片长于约 90 秒时，转写里「预告后面有料」的 open-loop 句（「后面我讲多数人搞错的部分」）列为**高风险删除候选**，不当口水话轻删——它是中段留存的钩子。对偶条款：若承诺的内容在成片后段并未出现（被剪掉或本就没讲），该开环句反而**优先删除**，避免空头支票。
- **受众路由**：25+ 专业受众（如 BI/数据从业者）或 B 站横屏长视频走慢档——约 20–40 秒一切，只在话题/语气明确转换处切，不套娱乐向快切教条。

## 结尾

- **默认自然结束**：在最后一个完整有效口述句与必要自然气口后直接结束。不自动添加模板片尾、品牌落板、循环回环、额外 CTA、人物淡出收束页或为了做淡出而延长空尾。
- **有条件保留**：只有用户明确要求，或原素材中已有必要结论/CTA 且下方平台路由决定保留时，才设计结尾结构。循环呼应同样是可选项；素材确有与钩子呼应的完整句子才使用，没有就不硬造。
- **淡出边界**：可在不吞最后音节时给主音频约 0.25–0.5 秒短淡出。视觉淡出只有在能由一个最终合成所有者整体接管时才做（见 `motion-transitions.md` 装饰层生命周期）。

## 数据复盘口径

发布后的平台数据是本文件参数的校正来源，对应关系：

| 回流指标异常 | 校正动作 |
| --- | --- |
| 2 秒/3 秒跳出率高 | 重跑「开头重构决策流」：检查第一句是否钩子、有无残留铺垫、首帧是否合规 |
| 完播率低 / 人均播放时长短 | 重估单片时长档与气口节奏；检查中段是否命中 30–45 秒注意力时钟而未处理 |
| 封面点击率低 | 重做封面并连带复查首帧构图 |

数据只做本次剪辑简报的参数排序参考；固化为永久规则须经用户确认，本文件不自动改自己。
